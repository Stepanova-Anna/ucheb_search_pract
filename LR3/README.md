# Лабораторная работа № 3

## Работу выполнила Степанова Анна ИВТ 3 курс 1 группа 2 подгруппа

**Тема:** Разработка плана эксперимента по проверке корректности и эффективности проектных решений

**Цель:** научиться разрабатывать план эксперимента для объективной проверки проектных решений в ИТ‑проектах: оценить их корректность (соответствие требованиям) и эффективность (производительность, надёжность, ресурсоёмкость).

**Задачи:** 

1)	выбрать тему исследования и сформулировать проектное решение;
2)	определить критерии оценки корректности и эффективности;
3)	разработать план эксперимента: цели, гипотезы, методы, метрики, условия;
4)	провести пилотный эксперимент (на ограниченном наборе данных);
5)	оформить отчёт с результатами и выводами.

Лабораторная работа № 2
Тема: Сравнение теоретических и эмпирических методов исследования 
Цель: провести сравнительный анализ теоретических и эмпирических методов исследования, применить их на практике для решения конкретной ИТ-задачи, проанализировать преимущества и ограничения каждого из используемых методов
Задачи: 
1)	изучить сущность, виды и особенности теоретических и эмпирических методов исследования;
2)	выбрать ИТ задачу и решить её, используя сначала теоретические, затем эмпирические методы;
3)	сравнить результаты, точность, временные затраты и применимость методов;
4)	сформулировать рекомендации по выбору метода для аналогичных задач.
